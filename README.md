<h4>First Flask Deployed personal API</h4>
# Personal API

My first Flask API - returns information about me in JSON format.

## Live URL
https://flask-personal-api.onrender.com

## Endpoints

- `/info` - Basic information
- `/portfolio` - Portfolio details
- `/contact` - Contact information
- `/projects` - List of projects
- `/all` - All information combined

## Tech Stack
- Python
- Flask
- Deployed on Render

- Flask routing and responses
